# Cold Open Creator - Reference Materials

This folder contains supporting materials for the `cold-open-creator` skill.

## Files in This Folder

### `gold-markers-guide.md`
Complete searchable list of phrases and patterns to find in transcripts. Use this as your "fishing guide" when mining transcripts for dramatic moments.

### `technical-specifications.md`
Precise rules for timing, audio quality, visual language, and cut points. Reference when refining your cold open in Phase 3.

### `quality-control-checklist.md`
Detailed scoring rubric for the 5 quality tests. Use this to objectively evaluate whether your cold open creates sufficient psychological pull.

## How to Use These References

**During Phase 1 (Mining):** Keep `gold-markers-guide.md` open while scanning transcript

**During Phase 2 (Construction):** Reference when selecting which clips to include

**During Phase 3 (Refinement):** Use `technical-specifications.md` and `quality-control-checklist.md` to polish
